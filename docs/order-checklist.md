# Чеклист по заказу (лендинг + Метрика)

## От заказчика

- [ ] Картинка для favicon / иконки в поиске (можно через [realfavicongenerator.net](https://realfavicongenerator.net/))
- [ ] ID целей в Яндекс.Метрике (список: что за цель → номер)

## Сайт

- [ ] Текст на второй странице перед вторыми кнопками (заголовок + абзац — как в переписке)
- [ ] Логотип ВК — белый / в тон сайта
- [ ] Favicon подключён

## Метрика

- [ ] Счётчик на сайте
- [ ] Вебвизор включён
- [ ] Цели на кнопки «Макс» и «ВК» (по ID от заказчика)
- [ ] Кратко написать заказчику: где смотреть визиты, время, цели (подписку с сайта не видно — только клик)

## Сдача

- [ ] Всё проверить на домене заказчика
- [ ] Отдать на проверку после картинок и целей
