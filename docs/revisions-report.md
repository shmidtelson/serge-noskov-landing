# Что сделано (custom-landing)

- Одна тема оформления — **aurora**, остальные темы и страница сравнения вариантов убраны.
- Тексты героя, блок метрик (3 колонки), финальный блок перед кнопками — по согласованию.
- Кнопки **Подписаться** на **Max** и **ВК**, ссылки по умолчанию + из `.env`.
- Иконки Max и ВК — inline-компоненты; ВК белый, на светлой кнопке финала — через `currentColor`.
- Яндекс.Метрика: счётчик, **вебвизор** в `init`, цели на клики по кнопкам (`reachGoal`, `data-ym-goal`).
- Favicon и **webmanifest** из папки `public/favicons/`, подключение в `Base.astro`.
- Файл **`.env.example`** — Max, ВК (в коде), Метрика (`PUBLIC_YM_*`).
- Документ **`docs/order-checklist.md`** — короткий чеклист по заказу.
